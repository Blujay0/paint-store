# Paint Shop
Online storefront for a home improvement store. <br />
Playground for experimenting with styling + page layout.

## Features
* HTML
* Semantic HTML
* CSS
* CSS specificity
* rgba()
* hsla()
* @media queries
* Google Fonts API
